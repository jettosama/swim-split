How to use

01
1. Load a video. Choose length + start mode.
2. Scrub to the start frame, tap MARK START.
3. Scrub to each checkpoint as the head crosses it (15m flag, T-mark, 35m flag, wall), tap MARK.
4. Optional: Play the clip through (slow if needed) and tap S each time an arm enters the water. SR, DPS, and Stroke Index compute automatically per segment.

Shortcuts: M mark · S stroke · Space play · ← → frame · Z undo mark · X undo stroke
